기본에 충실한 신입 백엔드 개발자

기본에 충실한 협업
최고의 팀은 기본에 충실합니다. 기본적인 문서화와 팀의 개발 규칙을 준수하여 팀원들과 효율적으로 소통합니다. 좋은 코드를 작성하기 위해 노력하고 상황 공유를 통해 지속적으로 피드백을 주고 받으며 더 나은 결과를 만들어 갑니다.

시간 관리로 완성되는 모든 기본
시간 관리는 모든 성공의 근본입니다. 우선순위를 명확히 설정하고 업무를 효율적으로 배분하여 정해진 시간 내에 목표를 달성합니다. 팀원으로서는 맡은 역할을 충실히 수행하고, 리더로서는 업무를 효율적으로 조율하여 팀의 목표를 달성하는 데 기여합니다.

기초를 통한 성장
탄탄한 실력을 쌓기 위해 이론을 깊이 있게 학습하고 이를 바탕으로 실무에서 발생하는 문제를 해결할 수 있는 능력을 키웁니다. 점차적으로 고급 기술과 개념을 익히고 응용력과 문제 해결 능력을 지속적으로 발전시켜 나갑니다.

SkillSet

BackEnd

  • Java
    Java의 기본 문법을 이해하고 JAVA에서 제공하는 클래스와 메소드를 적절하게 사용할 수 있습니다. JAVA를 이용하여 콘솔 기반의 은행 프로그램을 제작한 경험이 있습니다.
  • JSP
    JSTL과 EL 표현식을 활용할 수 있습니다. Model2 방식을 기반으로 회원 인증과 게시판 기능, 쿠키를 이용한 아이디 저장 기능, AJAX를 활용한 좋아요 기능, SMTP로 이메일 전송을 통한 비밀번호 찾기 기능을 구현한 경험이 있습니다.
  • Spring
    DI와 IoC를 이해하고 어노테이션 기반 Controller를 구현할 수 있습니다. Spring MVC 패턴을 이용하여 동적 웹페이지를 제작할 수 있습니다.
  • MyBatis
    XML 기반으로 SQL문을 작성하고 데이터베이스 연동을 구현할 수 있습니다.

FrontEnd

  • HTML5
    시멘틱 마크업을 작성할 수 있습니다.
  • CSS
    기초적인 UI 설계와 기본 문법을 숙지하고 있습니다. 부트스트랩을 활용한 반응형 레이아웃 구현이 가능하며 브라우저 간 호환성을 고려한 작업을 수행할 수 있습니다.
  • JavaScript
    ES6 문법을 활용한 코드 작성이 가능하며, jQuery와 AJAX를 사용하여 비동기 처리와 동적 웹 페이지를 구현할 수 있습니다.
  • React
    React를 사용하여 컴포넌트 설계와 상태 관리 그리고 페이지 라우팅 및 렌더링 최적화를 구현할 수 있습니다. Firebase를 활용하여 채팅 앱을 구현한 경험이 있습니다.

Mobile

  • Dart
    Dart의 기본 문법을 이해하고 모바일 애플리케이션 개발을 할 수 있습니다. Flutter 프레임워크를 이용한 지역기반 의료 서비스 매칭 애플리케이션 개발 경험이 있습니다.
  • Flutter
    다양한 위젯을 커스텀하여 UI를 제작하고 Provider로 효율적인 상태 관리를 할 수 있습니다. HTTP 요청을 통해 API와 통신하여 서버에서 데이터를 가져와 구동하는 지역기반 의료 서비스 매칭 애플리케이션을 제작한 경험이 있습니다.
  • Android Studio
    Emulator를 사용한 다양한 Android 버전과 기기에서의 앱 테스트 경험이 있습니다.
